# Java RPG Game
Um jogo do gênero RPG feito para a disciplina de Lógica Matemática da faculdade, usando Java. O projeto aplica os conceitos de proposições lógicas, onde cada escolha do jogador representa um valor lógico.

## Como Jogar?
* O jogo funciona totalmente no terminal.
* Digite o número ou comando de acordo com a sua ação e pressione `Enter`.

## Tecnologias
* Java 24

## Como Rodar o Projeto?
**Pré-requisitos**  
* É necessário ter instalado o **Java SDK 24**.

1. Clone o repositório
```bash
git clone https://github.com/LeviSantos000/java-rpg-game.git
cd java-rpg-game
```
2. Execute o arquivo
```bash
java Jogo.java
```
